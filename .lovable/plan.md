

## שיפור כרטיסי "מה לומדים בקורס" – אפקט hover עדין

### שינוי יחיד: `src/components/landing/CourseContentSection.tsx`

**אפקט hover על הכרטיס:**
- `scale(1.04)` + `translateY(-4px)` + צל חזק יותר
- `transition: all 0.3s ease`

**טקסט תיאור נסתר:**
- פסקה קצרה בעברית מתחת לכותרת כל כרטיס
- ברירת מחדל: `opacity-0`, בהובר: fade-in + slide-up קל
- ללא שינוי layout (שימוש ב-absolute positioning או max-height)

**מיפוי טקסטים (עברית בלבד):**

| כותרת | תיאור |
|---|---|
| בניית מחמצת מאפס | תהליך ברור ומדויק מהיום הראשון ועד מחמצת פעילה ויציבה. |
| האכלות ותסיסות | נבין איך לקרוא את המחמצת ולדעת בדיוק מתי היא בשיאה. |
| קיפולים נכונים | נלמד כיצד לפתח רשת גלוטן חזקה שתעניק ללחם נפח ואווריריות מושלמת. |
| עיצוב וכדרור | נלמד לעצב בצק מתוח וחלק לפני ההתפחה הסופית. |
| חריצה מקצועית | טכניקות חריצה שיאפשרו ללחם להיפתח יפה ומבוקר באפייה. |
| אפייה בתנור ביתי | נלמד ליצור אדים נכונים ולהפיק תוצאה מקצועית גם בתנור ביתי. |

### פרטים טכניים
- הוספת שדה `description` למערך `courses` הקיים
- הוספת Tailwind classes לכרטיס: `hover:scale-[1.04] hover:-translate-y-1 hover:shadow-lg transition-all duration-300 ease-in-out`
- הפסקה תקבל: `opacity-0 group-hover:opacity-100 translate-y-2 group-hover:translate-y-0 transition-all duration-300`
- הכרטיס יקבל class `group` לתמיכה ב-group-hover
- אין שינוי בקבצים אחרים, אין רכיבים חדשים, אין שינוי צבעים
